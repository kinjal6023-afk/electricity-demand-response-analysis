# Electricity Demand Response & Supply–Demand Analysis

## Project Overview
This project analyzes electricity supply and demand patterns in power distribution networks using SQL. The goal is to understand hourly electricity demand fluctuations and explore how demand response strategies help balance electricity consumption in the grid.

## Tools Used
- SQL (SQLite)
- DB Browser for SQLite
- Data Analysis
- PowerPoint Visualization

## Dataset
The dataset contains hourly electricity demand data from the NYISO electricity market.

Main dataset file:
nyiso_hourly_loads.csv

The data was stored and analyzed using a SQLite database.

## Key Analysis Performed
• Hourly electricity demand analysis  
• Identification of peak demand hours  
• Understanding electricity consumption trends  
• Exploring demand response strategies to reduce grid pressure

## Key Insights
- Electricity demand varies significantly throughout the day.
- Peak demand periods create pressure on electricity distribution systems.
- Demand response strategies can help stabilize electricity supply and improve grid efficiency.

## Project Files
- electricity_market.db – SQLite database used for analysis
- nyiso_hourly_loads.csv – raw dataset
